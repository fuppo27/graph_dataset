# graph_dataset
## format
* edge.csv: edge information file
  * node_ids are from 0 to #nodes-1
  * head: None
  * body: from_node_id,to_node_id  
* label-list: edge information file
  * node_ids are from 1 to #nodes
  * head: #nodes #nodes 1
  * body: degree node_ids
